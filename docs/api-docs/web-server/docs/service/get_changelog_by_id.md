### 描述

- 该接口提供版本：v9.9.9+。
- 该接口所需权限：无。
- 该接口功能描述：获取指定版本日志。

### URL

GET /api/v1/web/changelog/{id}


### 输入参数

| 参数名称 | 参数类型   | 必选 | 描述   |
|------|--------|----|------|
| id   | string | 是  | 版本id |


### 响应示例

```json
{
  "code": 0,
  "message": "",
  "data": "## 优化\n\n"
}
```

### 响应参数说明

| 参数名称    | 参数类型   | 描述   |
|---------|--------|------|
| code    | int32  | 状态码  |
| message | string | 请求信息 |
| data    | string | 响应数据 |
